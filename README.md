# 🌴 Wisata Kebumen App

Aplikasi mobile berbasis Android yang digunakan untuk membantu pengguna menemukan berbagai tempat wisata di Kebumen dengan mudah.  
Aplikasi ini menyediakan informasi wisata seperti deskripsi, kategori, jarak dari lokasi pengguna, serta fitur navigasi langsung ke lokasi menggunakan Google Maps.

---

## 📱 Fitur

- 🔍 Pencarian wisata berdasarkan nama
- 🗂️ Filter kategori (Pantai, Alam, Sejarah)
- ❤️ Menyimpan wisata favorit
- 📍 Menampilkan jarak dari lokasi pengguna
- 🗺️ Lihat lokasi di Google Maps
- 🚗 Navigasi langsung ke tempat wisata
- 🔐 Mode Admin (tambah, edit, hapus data wisata)

---

## 🚀 Cara Install

1. Clone repository ini: https://github.com/wongsxs/WisataKebumen
2. Buka project di Android Studio
3. Tunggu proses Gradle selesai
4. Jalankan aplikasi di emulator / device

---

## 🛠️ Teknologi

- Kotlin
- Android Studio
- Firebase Realtime Database
- Google Maps Intent (Navigation)
- Location Services (GPS)
